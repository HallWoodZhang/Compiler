# Compiler UI

## For glut environment
### 执行以下步骤
1. 打开glut文件夹
2. 将glut.h放到x:\你电脑VS的安装路径\include\GL文件夹下，如果没有 GL 文件夹,就创建一个.
3. 将glut64.lib放到：x:\你电脑VS的安装路径\lib\x64 下
4. 将glut64.dll放到：x:\Windows\System32
5. 重新构建项目即可
