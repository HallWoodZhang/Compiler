# 团队开发中用到的一些 Git 基本操作

## git log
查看版本日志

## git status
查看当前分支状态

## git add
添加当前某个文件的修改

## git commit -m "message"
提交

## git push (仓库 分支)
推到自己的远程仓库


## git fetch XXX 分支
拉取一个人的一个分支

## git rebase FETCH_HEAD
应用 FETCH_HEAD

## git reset --hard HEAD
版本回退

##  发起 Pull Request



## git remote -v
显示远端仓库

## git remote add XXX Url
添加远端仓库
